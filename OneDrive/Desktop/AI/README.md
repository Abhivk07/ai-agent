# AI Agent

A simple AI agent built with Python using OpenAI's GPT model.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Set up your OpenAI API key in `.env` file.

3. Run the console version:
   ```
   python src/main.py
   ```

4. Run the web version:
   ```
   python web.py
   ```
   Then open http://127.0.0.1:5000 in your browser.

## Features

- Interactive chat with AI via console or web
- Uses GPT-3.5-turbo model
